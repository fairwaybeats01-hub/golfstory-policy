골스토리 계정 삭제 안내

골스토리 앱에서 계정 및 데이터를 삭제하는 방법을 안내합니다.

앱 내 직접 삭제 방법

1단계: 골스토리 앱을 실행합니다.
2단계: 하단 메뉴에서 프로필을 선택합니다.
3단계: 상단 우측 설정(⚙️) 아이콘을 선택합니다.
4단계: 회원탈퇴 메뉴를 선택합니다.
5단계: 탈퇴 확인 후 계정 및 모든 데이터가 즉시 삭제됩니다.

삭제되는 데이터

계정 삭제 시 아래 데이터가 즉시 영구 삭제됩니다.

• Firebase 계정 정보 (익명 UID)
• 성별, 14세 이상 여부
• 출발지 좌표 및 주소
• FCM 푸시 알림 토큰
• 골프 라운드 기록 (스코어, 홀 정보 등)
• 예약된 라운드 정보
• 푸시 알림 데이터

데이터 보관 기간

모든 데이터는 서비스 이용 기간 동안만 보관되며, 계정 삭제 즉시 파기됩니다. 별도의 보관 기간은 없습니다.
